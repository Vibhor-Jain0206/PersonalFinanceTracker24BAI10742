# PersonalFinanceTracker24BAI10742

## Description

This is a simple Java-based console application that helps users track their daily expenses. It allows users to add expenses, view all recorded entries, and calculate the total amount spent.

## Problem

Many people do not keep a proper record of their daily spending, which can lead to poor financial management.

## Solution

This project provides a basic system to store and manage expenses in an organized way using Java.

## Features

* `Add expense` (category and amount)
* `View all expenses`
* `Calculate total expense`

## Concepts Used

* Classes and Objects
* ArrayList
* Loops and Conditionals
* Scanner for user input

## How to Run

1. Compile the program:
   ```
   javac Main.java
   ```
2. Run the program:
   ```
   java Main
   ```
